# gn-pmenu
A personal menu made for FiveM roleplay server using ESX framework. 

# Features:
- Legal Interactions, show id, car license and more.
- Illegal Interactions, bodysearch, arresting and more.
- Rockstar Editor
- Cinematic Camera Mode
